# FastPrompter для чайников (объясняет дед)

> Сел, чай налил? Хорошо. Слушай сюда, коротко и без соплей.

## Что это

Блокнот на одной клавише. Жмёшь `Alt+X` — окошко выскочило у мышки.
Жмёшь `Esc` — исчезло. Как кукушка из часов, только полезная. Внутри:
сто пронумерованных ячеек-«сило» под текст, снипеты-заготовки на
`F1`–`F10`, вкладки-проекты, ящик для файлов на каждую заметку, архив,
корзина. Всё это — портативная программа для Windows, ни установки,
ни админ-прав не просит.

## Для чего

Чтобы не терять мысль, пока лезешь за блокнотом. Промпт для нейросети,
кусок кода, список покупок, черновик письма — набросал, окно спряталось
само, текст сам сохранился. Кнопку «сохранить» жать не надо: перестал
печатать — оно уже легло на диск. Свет вырубили — банка цела.

## Почему так, а не в облаке

Потому что твой черновик — не чужое дело. Всё лежит в одной папке
`data/` рядом с программой: без аккаунта, без облака, без
телеметрии. Скопировал папку на флешку — вот тебе и бэкап, и вся
установка целиком.

## Куда что делось

- `data/local_data_v15.db` — база данных, пишется в реальном времени.
- `data/files/<проект>/<сило>/` — вложения по каждой заметке, обычные
  папки, открывай в проводнике когда хочешь.
- `data/files/_trash/` — сюда едет средним кликом удалённое: ничего не
  горит, всегда можно вытащить обратно.
- `Documents\.fastprompter\` — ежедневный markdown-слепок всего, на
  случай если сама программа вдруг не запустится — текст всё равно
  читается блокнотом.

## Как этим пользоваться (коротко)

- `Alt+X` — вызвать/спрятать окно из любой программы.
- `F1`–`F10` / `Ctrl+Shift+1`–`9` — вставить снипет от 1 до 10.
- `Ctrl+1`–`Ctrl+0` — прыгнуть на сило от 1 до 10.
- `Ctrl+N` — новое пустое сило; `Alt+↑`/`Alt+↓` — гулять по ним.
- `Ctrl+W` — вставить разделитель ---.
- `Alt+W` — вставить разделитель --- и начать новый маркер.
- `Ctrl+E` — сделать строку заголовком: # + жирный + подчёркнутый + время.
- `Ctrl+Enter` — переключить [ ] чекбоксы на строке/выделении.
- `Ctrl+B` / `Ctrl+I` / `Ctrl+U` / `Ctrl+T` — жирный, курсив, подчёркнутый, зачёркнутый текст.
- `Alt+Backspace` — удалить предыдущее слово.
- `Ctrl+S` — сохранить снипет.
- `Ctrl+D` — дзен-режим (фокус).
- `Ctrl+Q` — примагнитить окно к углам экрана.
- Средний клик по сило — в корзину (не насовсем).
- Наведи мышь на сило — вылезут кнопки: тик ✅, файлы 📁, пин 📌, архив 📥.
- Правой кнопкой по списку проектов — добавить, переименовать или удалить вкладку.

Полный список фич, скриншоты и разбор каждой кнопки — в
[README.md](README.md#инструкция--instruction), там отдельная
дедовская глава на русском и английском.

## Итого

Один хук — `Alt+X` — и весь твой хаос из мыслей, кода и ссылок живёт в
одном месте, никуда не убегает и никому не докладывает. Дед одобряет.
